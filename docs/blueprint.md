# **App Name**: BizFlow

## Core Features:

- Financial Dashboard: Display a dashboard with key metrics: total revenue, total expenses, and calculated profit. The dashboard will allow the user to input these values.
- Appointment Calendar: Implement a simple calendar interface to view and manage appointments. Allow adding, viewing, and deleting appointments. The calendar does not need to sync to any external calendar services.

## Style Guidelines:

- Primary color: Blue (#3498db) for professionalism and trust.
- Secondary color: Light gray (#ecf0f1) for backgrounds and subtle elements.
- Accent: Green (#2ecc71) for positive indicators like profit, and key actions.
- Clean, card-based layout for the dashboard to present information clearly.
- Use a standard calendar layout for easy navigation and understanding.
- Use simple, flat icons for metrics and calendar actions.

## Original User Request:
Build a basic ERP for small businesses with a dashboard showing revenue, expenses, and profit, plus a calendar for appointments.
